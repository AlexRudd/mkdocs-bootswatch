MkDocs Bootstrap Theme

This project provides themes for the different Bootswatch themes for the MkDocs project.

Installation

Install the theme with pip:

pip install mkdocs-bootswatch

After the theme is installed, edit your mkdocs.yml file and set the theme to one of the following bootwatch themes that have been installed.

  • amelia
  • cerulean
  • cosmo
  • cyborg
  • flatly
  • journal
  • readable
  • simplex
  • slate
  • spacelab
  • united
  • yeti

For example:

theme: amelia

Or:

theme: yeti
